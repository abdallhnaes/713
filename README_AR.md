# تطبيق العمران للخطط والديربي - V3

تطبيق أندرويد Native لرسم خطة فريق العمران بشكل قريب من التصميم المطلوب.

## الميزات
- أرضية ملعب بشكل واقعي: عشب مخطط ومربعات، خطوط ملعب، مدرج، جدران، إطار أحمر.
- خطة افتراضية 3-3-1.
- تعديل أسماء اللاعبين والبدلاء من داخل التطبيق.
- إظهار / إخفاء أسهم التحركات.
- حفظ الخطة كصورة في الهاتف داخل Pictures/AlOmran.
- قسم الديربي:
  - إضافة اسم الفريق المنافس.
  - إضافة نتيجة المباراة.
  - حفظ النتائج محليًا داخل التطبيق.
  - عرض التشكيلة الحالية مع نتائج الديربي.

## الأسماء الافتراضية
- الحارس: ابو الليث
- المدافعون: حسونة - عبدالله - عبدالخالق
- أمام المدافعين: جليبيب
- الوسط المتقدم: سلوم - ابو احمد
- رأس الحربة: يوسف
- البديل: مصطفى عثمان

## بناء APK من GitHub
1. ارفع كل ملفات المشروع إلى GitHub.
2. افتح تبويب Actions.
3. شغل Build APK.
4. حمل الملف من Artifacts باسم:
   AlOmranTacticsApp-v3-apk
